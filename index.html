Calendar Tasks

A task list that automatically pushes finished tasks to Google Calendar as past events.

How it works

Authorize access to Google Calendar API

Create new task lists by creating new calendars in Google Calendar. These become your task lists. (You may need to refresh a few times for them to sync.)

Add new tasks by tapping

Name or edit your tasks by tapping inside them.

To remove an unwanted task tap

To start a task tap

To finish a task and record it as an event in Google Calendar, tap again

Calendar Suggestions

SLEEP  - Start a task each night and complete it when you wake up. Use it to see your sleeping pattern throughout the week.
FOOD  - Start a task and complete it everytime you eat something. Use it as a food journal for dieting.
WORKOUT  - Start a task when you begin your workout and complete it when you finish. Use it as a fitness journal.
OTHER  - Use this task list for tasks or events that wouldn't logically fit in another category.
SCHEDULE  - Don't add tasks to this calendar. Instead, schedule your day with it in Google Calendars, then use other calendars to complete your tasks. At the end of the day compare what you scheduled with the tasks you completed.
